# API

NestJS service for the platform.

## Run locally

From repo root (after `bun install`):

```bash
bun run dev --filter api
```

Available at http://localhost:3001.

## Build

```bash
bun run build --filter api
```

## Test

```bash
bun run test --filter api
bun run test:e2e --filter api
```

## Lint

```bash
bun run lint --filter api
```
